
Документ состоит из 3 экранов: тегов div с классом screen.

1) Первый экран состоит из тега div c классом menu bar и обложки экрана. Тег div c классом menu bar содержит в себе div c id: left menu и div c id: right menu. Div c id: right menu содержит в себе 4 div с классом menu item.

 2) Второй экран состоит из div c классом article в котором находится div c классом box. В div c классом box содержится две статьи div с id:left text и div с id:right text.
 
3) Третий экран переставляет собой галлерею из 6 изображений. Div с классом gellery является родителем по отношению к div c классом img block. В теге div c классом img block  находятся шесть div с классом cell.
Заключительной частью сайта является footer, в котором находятся 3 flex-контейнера с дополнительной информацией и copyright автора. 

В селекторе menu bar прописываем свойство 
display со значением flex, задаем цвет фона через свойство background-color, а так же задаем размер шрифта(font-size).

Верстка сайта сделана через CSS Flexbox. Для этого я установила необходимому HTML элементу CSS-свойства  display со значением flex.
